## BDCatalyst-RFC-9 - Application Developer Guide 1.0

This document is intended to provide sufficient guidelines for an "app" developer and an "app" system launcher to build their respective resources in the BioData Catalyst. **In this context an "app" is a single or multi-container Docker-based application that typically has a web-based user interface.** It is intended to be launched in a computational environment in the cloud by an end user for their own use.
