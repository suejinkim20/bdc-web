_Closed. See [BDCatalyst-RFC-10: BioData Catalyst Minimal Metadata Model](https://www.biodatacatalyst.org/collaboration/rfcs/bdcatalyst-rfc-10/)_

BioData Catalyst is an ecosystem built upon different platforms, each of which contains a unique set of tooling to access, filter, and analyze data stored in NHLBI cloud buckets. Metadata (data about the data) is the primary currency that flows between these platforms and enables findability and interoperability. While these platforms may have different functions and use different pieces of metadata to different ends, it is imperative that there is an agreement on a fundamental, core set of metadata to ensure basic operation of the ecosystem. Thus, in this RFC a file-level minimal metadata model is proposed, comprising metadata elements which are required for indexing files into the ecosystem and for retrieval of files by a user with appropriate credentials.
