This document describes and defines data movement (data egress and ingress), explaining the types of data movement currently allowable on each platform, and what kinds of data should be downloaded or uploaded. This document is meant to inform the BioData Catalyst consortium of the Data Access Working Group’s (DAWG) recommendations for policy and language to be used to educate the Go Live BioData Catalyst users.
